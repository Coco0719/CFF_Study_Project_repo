# Cpp_Study_Project_repo


이 저장소는 제가 C++을 공부하면서 만들어본 작은 프로젝트들을 저장하기위한 저장소입니다.

내가 C++ 공부한방법!!

1. 유튜브에 "어소트락 게임아카데미 c++ 게임" 검색

2. 여기서 나온 영상에서 매번 초반에 만들것을 알려줌 >> 그러면 나는 영상을 정지하고 그 기능을 직접 구현해봄
   (저는 이미 C언어로 학교 텀프로젝트로 여러가지 과제를 하였기 때문에, 쉽게쉽게 빨리 구현할 수 있었음)

3. 영상을 보고 나와 무엇이 다른지, 강좌 찍으신 분은 어떤 사고과정을 거쳐서 저런 코드를 짯는지 사고를 흉내냄
   (나와 비슷한 사고를 한거 같으면 그냥 빨리빨리 넘어감)
   (내가 몰랐던 새로운 것이 있다면 집중해서 들음)

* 주의사항 : 제가 작성한 코드와 동영상에서의 코드는 다릅니다!!
